```
1、单从教材内容来说，《美国语文》系列比《新概念英语》系列要更适合作为学习教材。但新概念有一个强大的优势，几乎是中文世界独有的英语学习优势，基于新概念产生的学习资源实在是太丰富了。包括不限于：跟课文一一对应的精讲视频课、可以拿来当基础听力训练的音频、各种背单词App都针对新概念有专门的词库，以及各种学习笔记，学习经验，这是新概念至今仍被大家推荐给各个年龄段学英语的重要原因。

2、新概念的课文精讲视频很容易购买，网上也能搜到一些免费的老版本，以自己的学习效果检验，挑自己最喜欢的版本即可，优酷有免费的《裕兴新概念》作为最不坏的选择，Youtube上也有。

3、搭配视频课，先完整的过一遍，这是基础，也是最适合普通人的学习计划。看视频的同时，利用单词App辅助背一下单词，词库就设置成新概念1-3，按序重复。

4、从新概念2-4，适合做最基本的听力训练。学习过的课文，碎片时间多听，慢慢锻炼听力。新概念3更是专门为了“写作”板块撰写的课文。熟能生巧，多读，仿写课文，利用新概念的基础做基础的写作训练，有兴趣的甚至可以背一背课文。缺乏基本写作能力的适合，背喜欢的课文、段落，再按照自己的兴趣改写，十分有效果。
```

```

1、理论部分我们都知道掌握一门语言是由听、说、读、写四个部分来组成的。这四个部分也是有先后顺序的，我们应该遵守，先听，后说，再读写这样的顺序进行，这其实也正是我们学习母语的步骤。看看我们学汉语的过程：10个月时，开始意识听到声音1岁开始意识到语言与事物的联系1岁半只会说单个的名词3岁左右完全学会了说话6岁开始简单的阅读、识字10岁开始简单的写作首先是大量听，然后是自然说，接着写字、句子，最后写作，这就是母语学习的过程！学英语也是相似的，但是，我们从小所接受的应试英语教育过度强调书面的读和写，而忽视了听和说，这导致很多中国人的英语都停留在哑巴英语的阶段。新概念英语好就好在它的每一篇文章都是为了提升英语听说读写四个方面的能力而设计。也就是，理论上，你可以利用新概念英语来同时提升听力，口语，阅读和写作。


2、实操部分新概念英语教材最好按照1234的顺序学，很多人认为第一册比较简单，就直接跳过，其实第一册也是非常重要的。一册重语音；二册重语法；三册重词汇+写作；四册重阅读，每种教材都涉及到了不同的题型，所以只有前面的基础打牢了，后面学习才会更轻松。首先，当我们拿到一篇课文时，要先听写。先听3-5遍，大致理解一下意思，然后再一句一句地听写，重复播放，直到能写下那句话。实在没有听出来的地方不建空着，而是听到什么写下什么，哪怕是几个零散的字母。结束后，对照课本找出自己听错的地方，找出原因再多听几遍，直到完全习惯它的发音。其次，听写完成后，接着就是记单词、词组和句型。一定要在这一步积累大量的词汇和句型，以后面的复述做准备。比如：by the way 顺便提一下，另外；on the whole 总的来说；Go to the theatre, 去看戏；Got angry, 生气等等，记住了简单的句型后，就需要自己慢慢跟着造句了。再接着就涉及到主动的口语表达了。可以用自己的话来复述原文，刚开始肯定是磕磕绊绊，支支吾吾的，说起来特别费力，甚至都不知道自己在说什么。因为是复述，虽然只能说出断断续续的句子，但只要能清楚表达出意思就已经很可以了。只要坚持练习，慢慢就能感受到用英语表达的一个概念了。最后，则就是一句一句地跟读音频了。当听完一句话的时候，你要能完完整整跟着音频读出来，如果听不懂就放慢速度，反复听，然后重复练习，直到那句话能读准确。一开始可能舌头会打结跟不上，没关系，不用太着急，慢慢来，因为有时甚至一句话要练几十遍上百遍，等这句话练熟了再开始下一句，以此重复练习。
```

```

新概念教材一共4册，内容比较经典也比较实用。从1册到4册难度逐渐增加，如果应对四六级英语考试基本上掌握透1-3册没什么问题。目前各个年龄段都有很多人在学习这套教材。以下是1-3册的学习内容，可以参考下。选择适合自己的课程也是非常重要的。

新概念一册学习范围：小升初词汇和考点，中考知识点学习内容：
① 1-30：讲解名词、代词、be动词、实义（行为）动词、形容词、冠词，学习a/an/the的区别和用法，学习there be、肯否疑问句、be动词基础上的特殊疑问句、祈使句、any/some等词的用法；
② 31-72：讲解时态，如一般现在时、现在进行时、一般将来时（be going to）、一般过去时，5个简单句结构；
③ 73-108：讲解中考考查的现在完成时，一定会考的宾语从句，以及一般过去时的不规则用法，副词的用法、复合句等；
④109-144（中考知识点）：讲解更多复合句，比较级/最高级的特殊用法，宾语从句时态等。

新概念二册学习范围：中考知识点，高考知识点学习内容：
① 1-24：讲解将来进行时、简单句结构、并列句等，句型结构与时态穿插；
② 25-48：时态加入复合句，讲解9种状语从句、1种定语从句、4种名词性从句；
③ 49-72：讲解并列句、简单句、复合句、非谓语动词、被动语态等；
④ 73-96：复习课，将1-72每课知识点融合，建立牢固的英语知识结构和语法体系。

新概念三册学习范围：国内高考、四六级等知识，托福、SAT等国外考试知识点学习内容：
① 1-20：讲解千变万化的“关键句型”，非谓语动词的灵活运用，较高级语法中的独立主格，倒装强调句式，双重否定等语法重难点；
② 21-40：讲解复杂的写作技巧和简化句子的方式，灵活运用长短句交替搭建文章框架，如并列句式，经典下定义，列举论据句型等；
③41-60：词汇量篇幅长度阶梯式增长，背诵仿写经典篇章。
```

```
对于《新概念》的学习，首先要告诉你的是一定要下苦功夫，不能嫌烦，坚持下去。而且要从第一册开始学起，这样对于你以后的学习会有好处的。

相对来说，第一册简单一些，基本上是一些日常对话，或者简单的叙述性短文。别看简单，如果你能脱口而出的话，就已经相当了不起了，而且，对于第二、三、四册的改写范文，能够打下坚实的基础。

背诵二、三、四册全文，理解性的去记忆，无须一字不落，你学英文的目的是为了能够表达思想，而不是单纯的背诵。机械式的背诵只能让你打击自信，失败得更快。要理解性地去背诵，当明白课文的意思时，就要学会去改写它，
把范文中所有的文章改写为自己的事情，让自己成为第一人，而不是一个阅读者。这样，你的记忆才能深刻，才能积累语言，迅速地表达思想。

```
